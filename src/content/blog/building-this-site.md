This is the first post on my new site. I designed the look in Figma, then turned it
into a real website I can update myself — no page-builder lock-in.

## Why bother?

A portfolio should be easy to update. Every time I finish a project, I want to add it
in a few minutes, not fight a clunky editor. So the goals were simple:

- **Full control** over how every page looks
- **Easy content updates** — just edit a file and the site updates
- **Room to grow** — projects today, blog posts (like this one) going forward

## How it works

Projects live as structured data, and posts like this one are written in **Markdown**.
That means I can use familiar formatting:

1. Numbered lists
2. **Bold** and *italic* text
3. [Links](https://example.com)
4. Code, quotes, and images

> The best portfolio is the one you'll actually keep up to date.
